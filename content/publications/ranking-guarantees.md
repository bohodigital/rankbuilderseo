## Interrogate the noun

What exactly is being guaranteed: one branded query, a set of commercial terms, local-map visibility, traffic, leads, or revenue? Each is a different claim.

## Inspect the measurement

Rankings vary by location, language, device, time, personalization, and result type. A guarantee without a measurement protocol is mostly theater.

- Named query set

- Location and device

- Measurement platform

- Start and end dates

- What happens on failure

## Read the escape hatches

Look for exclusions tied to approvals, developer capacity, algorithm changes, competitors, site migrations, or content access. Some agreements guarantee only that work continues—not that money returns.

## Buy controllable work

Providers can responsibly promise deliverables, response times, documentation, access, and testing discipline. They cannot control an external ranking system.
